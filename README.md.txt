# ПрАТ "Завод АКЦЕНТ" — сайт

## Развёртывание на GitHub Pages
1. Создайте новый репозиторий, например `akcent`.
2. Добавьте файлы `index.html`, `style.css`, `README.md` и папку `img/` с файлом `logo.png`.
3. Выполните команды:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/<ваш-логин>/akcent.git
   git push -u origin main
